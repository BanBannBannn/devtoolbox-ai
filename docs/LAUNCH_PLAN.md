Launch the first version of DevToolBox AI quickly, then improve based on traffic and feedback.

- Create project
- Add layout, header, footer
- Add home page
- Add tools page

Day 2:
- Build JSON Formatter logic and tests
- Build JSON Formatter UI

Day 3:
- Build README Generator logic and tests
- Build README Generator UI

Day 4:
- Build AI Coding Prompt Generator
- Add basic SEO content to first 3 tools

Day 5:
- Add About, Contact, Privacy Policy, Terms
- Add sitemap and robots.txt

Day 6:
- Deploy
- Fix mobile layout
- Fix broken links

Day 7:
- Add Google Search Console
- Submit sitemap

## Week 2: Content
- Write 5 supporting articles.
- Add FAQs to tool pages.
- Improve internal links.
- Improve page titles and descriptions.

## Week 3: More Tools
- Build Git Command Helper.
- Build Test Case Checklist Generator.
- Add 5 more articles.

## Week 4: Monetization Preparation
- Complete AdSense checklist.
- Improve UX and performance.
- Add more original content.
- Apply for AdSense only when the site is ready.

## Traffic Plan
- Share tools in developer communities without spamming.
- Create GitHub repo and link the website.
- Make short videos showing each tool.
- Write SEO articles around each tool.
- Keep improving based on Search Console queries.
