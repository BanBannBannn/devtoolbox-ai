# JWT Decoder QA Checklist

## Manual QA
- [ ] Page loads successfully.
- [ ] User can paste a valid JWT.
- [ ] Header decodes correctly.
- [ ] Payload decodes correctly.
- [ ] Signature segment displays when present.
- [ ] Empty input shows a helpful error.
- [ ] Malformed JWT shows a helpful error.
- [ ] Invalid Base64URL or JSON shows a helpful error.
- [ ] Timestamp claims display when present.
- [ ] Page clearly states that signatures are not verified.
- [ ] Page works on mobile.
- [ ] Page has SEO content and FAQ.
