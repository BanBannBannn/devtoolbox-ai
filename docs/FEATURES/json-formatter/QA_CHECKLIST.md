# JSON Formatter QA Checklist

## Manual QA

### Basic
- [ ] Page loads successfully.
- [ ] Textarea is visible.
- [ ] Format button is visible.
- [ ] Copy button is visible.
- [ ] Clear button is visible.

### Functionality
- [ ] Valid object formats correctly.
- [ ] Valid array formats correctly.
- [ ] Nested object formats correctly.
- [ ] Invalid JSON shows an error.
- [ ] Empty input shows an error.
- [ ] Clear button clears all state.
- [ ] Copy button copies output.

### UX
- [ ] Error message is easy to understand.
- [ ] Output is readable.
- [ ] Page works on mobile.
- [ ] Buttons are easy to tap.
- [ ] No horizontal scroll.

### SEO Content
- [ ] Page has one H1.
- [ ] Page explains what the tool does.
- [ ] Page includes example input and output.
- [ ] Page includes FAQ.
- [ ] Page links to related tools.
