# Date Calculator QA Checklist

- [ ] Page loads successfully.
- [ ] User can add days to a date.
- [ ] User can subtract days from a date.
- [ ] User can add months to a date.
- [ ] User can add years to a date.
- [ ] User can calculate days between two dates.
- [ ] Invalid date input shows a helpful error.
- [ ] Results use `YYYY-MM-DD` date format where appropriate.
- [ ] Page works on mobile.
- [ ] Page has SEO content and FAQ.
