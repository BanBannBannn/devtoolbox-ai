# QR Code Generator QA Checklist

## Manual QA
- [ ] Page loads successfully.
- [ ] User can enter plain text.
- [ ] User can enter a URL.
- [ ] User can generate a QR code.
- [ ] User can choose QR size.
- [ ] User can choose error correction level.
- [ ] Empty input shows a helpful error.
- [ ] Generated QR code is visible and scannable.
- [ ] Download PNG works.
- [ ] Clear button resets input and output.
- [ ] Page works on mobile.
- [ ] Page clearly explains what a QR code is.
- [ ] Page explains QR code size.
- [ ] Page explains error correction levels.
- [ ] Page includes common use cases.
- [ ] Page includes FAQ.
- [ ] Page includes related tools.
- [ ] No backend, database, AdSense, auth, or external AI API is added.
