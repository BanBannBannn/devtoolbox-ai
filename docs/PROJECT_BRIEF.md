# Project Brief

## Product Name
DevToolBox AI

## One-line Description
A fast, SEO-friendly website that provides free developer tools, AI coding prompts, and software testing checklists.

## Goal
Build a mostly-static Next.js website that can be deployed for free or cheap, attract organic traffic from Google, and later monetize with Google AdSense.

## Target Users
- Beginner software engineers
- Students learning programming
- Developers using AI coding tools
- People who need quick developer utilities

## Core Value
Users can solve small developer tasks instantly without login.

## Monetization Plan
### Phase 1
Google AdSense after the site has enough original useful content.

### Phase 2
Affiliate links for developer tools, hosting, domain services, or learning resources.

### Phase 3
Paid downloadable templates, prompt packs, testing sheets, or project starter kits.

## Product Constraints
- No login in v1.
- No database in v1.
- No paid APIs in v1.
- No backend unless absolutely necessary.
- Most tools should run fully in the browser.
- The website must be mobile-first.
- The website must be SEO-friendly.
- Each tool page must include useful written content, not only the tool UI.
- Do not use fake data that looks like real user data.
- Do not add unnecessary features.

## Success Metrics
- Website deployed successfully.
- At least 5 useful tools.
- At least 10-20 SEO pages or articles.
- Google Search Console connected.
- Sitemap submitted.
- AdSense checklist completed.
- First organic impressions from Google.
